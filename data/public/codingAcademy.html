scratch projects!

Scratch is a visual programming language that allows students to create their own interactive stories, games and animations. As students design Scratch projects, they learn to think creatively, reason systematically, and work collaboratively.
Block Coding is a style programming and it is very easy for anyone to learn! 

I can teach all ages from 6 to 18. Any level of programming is accepted for learning scratch. 

In Section 101, each project takes about 1 to 2 session each.

In Section 202, each project takes about 2 to 4 session each.

Each session is 60 dollars
and I have very flexible hours for scheduling 

Any project seen below can be twisted into any topic or genre to enjoy. If the student likes animals, then we can make a game with animals. This makes projects work with each individuals' enjoyed topics!
